🌦️ Weather Application

Описание проекта

Weather Application — современное клиентское веб-приложение для получения и отображения актуальной информации о погоде с использованием OpenWeather API.

Приложение позволяет пользователю получать данные о текущей погоде, просматривать почасовой прогноз и прогноз на 5 дней, использовать геолокацию для автоматического определения местоположения, а также сохранять пользовательские настройки.

Проект разработан в рамках дисциплины «Структуры данных и алгоритмы».

---

Основной функционал

Текущая погода

Приложение отображает:

- температуру воздуха;
- описание погодных условий;
- влажность;
- скорость ветра;
- погодную иконку.

---

Поиск по названию города

Пользователь может ввести название любого города и получить актуальные погодные данные.

---

Геолокация

Поддерживается автоматическое определение координат пользователя через Geolocation API браузера с последующим отображением погоды для текущего местоположения.

---

Почасовой прогноз

Отображается прогноз погоды на ближайшие часы с указанием:

- времени;
- температуры;
- погодных условий.

---

Прогноз на 5 дней

Для каждого дня выводятся:

- минимальная температура;
- максимальная температура;
- погодная иконка.

---

Темная и светлая тема

Реализована поддержка:

- Dark Mode;
- Light Mode;

а также автоматическое изменение цветовой схемы в зависимости от погодных условий.

---

Сохранение пользовательских данных

С использованием Local Storage сохраняются:

- последний выбранный город;
- выбранная тема оформления.

---

Обработка ошибок

Приложение корректно обрабатывает:

- отсутствие интернет-соединения;
- ошибки API;
- неверно введённые названия городов;
- отказ пользователя в предоставлении геолокации.

---

Используемые технологии

Frontend

- HTML5

- JavaScript (ES6+)

API

- OpenWeather API
- Geolocation API

Возможности браузера

---

Алгоритм работы приложения

1. Пользователь вводит название города или использует геолокацию.
2. Приложение отправляет запросы к OpenWeather API.
3. Сервер возвращает данные в формате JSON.
4. Полученные данные обрабатываются JavaScript-кодом.
5. Информация отображается в пользовательском интерфейсе.
6. Настройки пользователя сохраняются в Local Storage.

- асинхронное программирование;
- работа с REST API;
- обработка структурированных данных;
- работа с DOM-деревом;
- событийно-ориентированное программирование;
- локальное хранение данных;

---

Оптимизация производительности

Для повышения производительности реализованы:

- параллельные запросы через Promise.all();
- оптимизированное обновление DOM;
- локальное хранение данных пользователя;
- Skeleton Loading для улучшения пользовательского опыта;
- динамическая смена тем оформления.

---

Практическая ценность проекта

Проект демонстрирует практическое применение современных веб-технологий для взаимодействия с внешними API, обработки данных и создания интерактивных пользовательских интерфейсов.

Разработка позволяет закрепить навыки работы с JavaScript, асинхронными запросами, обработкой данных и построением клиентских веб-приложений.
