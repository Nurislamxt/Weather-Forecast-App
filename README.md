Weather Application

Описание проекта

Weather Application — это клиентское веб-приложение для получения и отображения погодных данных в режиме реального времени с использованием OpenWeather API.

Приложение позволяет пользователю получать текущую погоду, почасовой прогноз и прогноз на 5 дней для любого города мира, а также определять погоду по текущему местоположению пользователя.

---

Основной функционал

Текущая погода

- Температура воздуха
- Описание погодных условий
- Влажность
- Скорость ветра
- Иконка погодных условий

Поиск по городу

Пользователь может ввести название любого города и получить актуальные погодные данные.

Геолокация

Приложение поддерживает автоматическое определение местоположения пользователя через Geolocation API браузера.

Почасовой прогноз

Отображается прогноз погоды на ближайшие часы с указанием времени, температуры и погодных условий.

Прогноз на 5 дней

Показывается краткий прогноз на ближайшие пять дней с минимальной и максимальной температурой.

Сохранение данных

Последний выбранный город сохраняется в LocalStorage и автоматически загружается при следующем открытии приложения.

Темная и светлая тема

Пользователь может переключаться между светлым и темным режимом интерфейса.

Обработка ошибок

Приложение корректно обрабатывает:

- отсутствие интернет-соединения;
- ошибки API;
- неверно введённые названия городов;
- отказ в доступе к геолокации.

Используемые технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeather API
- Fetch API
- Geolocation API
- LocalStorage API

---

Алгоритм работы приложения

1. Пользователь вводит название города или использует геолокацию.
2. Приложение отправляет запрос к OpenWeather API.
3. Сервер возвращает данные в формате JSON.
4. Полученные данные обрабатываются JavaScript-кодом.
5. Информация отображается в пользовательском интерфейсе.
6. Последний выбранный город сохраняется в LocalStorage.

Особенности реализации

- Асинхронные запросы через async/await.
- Параллельное получение данных через Promise.all().
- Использование LocalStorage для хранения пользовательских настроек.
- Динамическое изменение темы в зависимости от погодных условий.
- Адаптивный интерфейс для мобильных устройств и компьютеров.
