🌦️ Weather Application

Описание проекта

Weather Application — современное клиентское веб-приложение для получения и отображения погодных данных в режиме реального времени с использованием OpenWeather API.

Приложение позволяет пользователю получать информацию о текущей погоде, просматривать почасовой прогноз, прогноз на 5 дней, определять местоположение через геолокацию, а также сохранять пользовательские настройки.

Проект разработан в рамках дисциплины «Структуры данных и алгоритмы».

---

🚀 Основной функционал

Текущая погода

Приложение отображает:

- текущую температуру воздуха;
- описание погодных условий;
- влажность;
- скорость ветра;
- иконку погодного состояния.

---

🔍 Поиск по городу

Пользователь может ввести название любого города и получить актуальную информацию о погоде.

---

📍 Геолокация

Поддерживается автоматическое определение местоположения пользователя через Geolocation API браузера.

---

⏰ Почасовой прогноз

Отображается прогноз на ближайшие часы с указанием:

- времени;
- температуры;
- погодных условий.

---

📅 Прогноз на 5 дней

Для каждого дня отображаются:

- минимальная температура;
- максимальная температура;
- погодная иконка.

---

🎨 Темная и светлая тема

Реализованы:

- Dark Mode;
- Light Mode;
- динамические цветовые схемы в зависимости от погодных условий.

---

💾 Сохранение данных

С использованием Local Storage сохраняются:

- последний выбранный город;
- выбранная тема оформления.

---

⚠️ Обработка ошибок

Приложение корректно обрабатывает:

- отсутствие интернет-соединения;
- ошибки API;
- неверно введённые названия городов;
- отказ в доступе к геолокации.

---

Frontend


Содержит структуру пользовательского интерфейса.

- адаптивную верстку;

Содержит бизнес-логику приложения:

- работу с OpenWeather API;

- сохранение пользовательских настроек.

---

⚙️ Алгоритм работы приложения

1. Пользователь вводит название города или использует геолокацию.
2. Приложение отправляет запросы к OpenWeather API.
3. Сервер возвращает данные в формате JSON.
4. Полученные данные обрабатываются JavaScript-кодом.
5. Информация отображается в пользовательском интерфейсе.
6. Пользовательские настройки сохраняются в Local Storage.

Для повышения производительности реализованы:

- динамическое изменение тем оформления.

---

🎯 Практическая ценность проекта

Проект демонстрирует практическое применение современных веб-технологий для взаимодействия с внешними API, обработки данных и создания интерактивных пользовательских интерфейсов.

Разработка позволяет закрепить навыки работы с JavaScript, асинхронными запросами, обработкой данных и построением клиентских веб-приложений.
